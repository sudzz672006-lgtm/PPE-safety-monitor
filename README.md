🏗️ PPE Compliance Monitor
This repository contains a Computer Vision-based safety monitoring tool. Using the YOLOv8 architecture, the application identifies personnel on construction sites and verifies if they are wearing the required safety gear.

Key Features:
Detection Classes: Helmets, Safety Vests, and Person.

Real-time Alerts: Instant visual feedback and error messaging for safety violations.

Multi-Source Support: Works with static image uploads and live webcam feeds.

Optimized Performance: Configured for low-latency inference on standard hardware.
